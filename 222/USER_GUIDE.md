# Инструкция пользователя

## Карта

Откройте вкладку **Атлас** и используйте фильтры слева: регион, район, карта-вопрос, поиск единицы и общий поиск. Городские пункты на карту не выводятся.

## Общий поиск и ареалы

Введите слово в поле **Общий поиск**. Приложение отфильтрует записи и построит ареалы по найденным лингвистическим единицам.

## Добавление населённого пункта

Во вкладке **Таблица** откройте блок **Добавить населённый пункт**. Заполните название, район и регион, затем нажмите **Найти координаты автоматически** или оставьте поля координат пустыми при добавлении записи — приложение попробует найти их само.

## Таблица

В отображаемой таблице не показываются столбцы «атлас», «год» и «источник». Номер вопроса выводится коротко: 1, 2, 3.
