# Интерактивная карта русских говоров Удмуртии

Streamlit-приложение для просмотра населённых пунктов, карт-вопросов, ареалов, изоглосс и таблицы диалектных данных.

## Что изменено

1. Городские точки скрываются на карте. В демо-данных оставлено 20 не городских населённых пунктов.
2. При использовании общего поиска ареалы строятся по найденной выборке.
3. Таблица очищена: в отображении убраны атлас, год и источник; номер вопроса отображается как 1, 2, 3. В демо-данных оставлено 3 вопроса ДАРЯ и 3 вопроса ЛАРНГ.
4. В блок добавления населённого пункта добавлен автоматический поиск координат: сначала локальный справочник, затем Nominatim / OpenStreetMap.
5. На карту добавлен слой границ регионов. При необходимости точный GeoJSON можно подключить через `BOUNDARY_GEOJSON_URL` в Streamlit secrets.

## Запуск

```bash
pip install -r requirements.txt
streamlit run streamlit_app.py
```

Основной файл для Streamlit Community Cloud: `streamlit_app.py`.

## Данные

Демо-таблица находится в `data/sample_dialects.csv`. Шаблон для новой таблицы — `data/data_template.csv`.

Одна строка = один населённый пункт + один вопрос.
