# Интерактивная карта русских говоров Удмуртии

Streamlit-приложение для просмотра населённых пунктов, карт-вопросов, ареалов, изоглосс и таблицы диалектных данных.

## Что изменено

1. Легенда перенесена в правую колонку карты и видна сразу, без нажатия.
2. Легенда сделана компактной: цвет, подпись категории и количество пунктов показываются в одной строке.
3. При первом входе, когда фильтры не заданы, все точки на карте отображаются одним цветом.
4. Основная таблица стала интерактивной и редактируемой через `st.data_editor`.
5. Для разных категорий точек используется контрастная палитра, чтобы цвета не были похожи друг на друга.

## Запуск

```bash
pip install -r requirements.txt
streamlit run streamlit_app.py
```

Основной файл для Streamlit Community Cloud: `streamlit_app.py`.

## Данные

Демо-таблица находится в `data/sample_dialects.csv`. Шаблон для новой таблицы — `data/data_template.csv`.

Одна строка = один населённый пункт + один вопрос.
